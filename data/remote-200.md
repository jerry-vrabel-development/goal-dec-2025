## How to Use This List

### Research Strategy
1. **Prioritize by remote policy:**
   - **Remote-first**: Company designed for remote work from the start
   - **Fully remote**: No physical offices, 100% distributed
   - **Remote-friendly**: Office exists but remote work fully supported
   - **Remote-flexible**: Mix of office and remote options

2. **Check company specifics:**
   - Visit careers page for current openings
   - Read engineering blogs for culture insights
   - Check Glassdoor/Levels.fyi for compensation
   - Verify time zone requirements (some need US/EU hours)

3. **Application approach:**
   - Remote jobs get 100+ applicants - differentiate yourself
   - Highlight remote work experience or async communication skills
   - Show you can work independently and manage time
   - Emphasize written communication abilities

### Remote Work Advantages for Senior Devs
- **Higher compensation** - Companies can pay well without geographic limits
- **Better work-life balance** - No commute, flexible schedules
- **Global opportunities** - Access companies worldwide
- **Async culture** - Often better for deep work
- **Location independence** - Work from anywhere

### Red Flags to Watch For
- "Remote for now" (planning return to office)
- Requires same timezone presence for 8+ hours
- No async communication culture
- All meetings, no documentation
- Micromanagement tendencies

### Key Remote Skills to Highlight
- Experience with remote collaboration tools (Slack, Zoom, Notion)
- Strong written communication
- Self-directed work habits
- Async workflow experience
- Time management across distributed teams
- Documentation mindset

### Geographic Considerations
- **US-only**: Many companies hire only US-based for legal/tax reasons
- **Americas timezone**: Common requirement for collaboration
- **Global**: True work-from-anywhere (rarer, but best flexibility)
- **Specific regions**: Europe, LATAM, Asia-Pacific hiring hubs

### Compensation Notes
- Remote companies often pay well to compete for talent
- Some use location-based pay, others pay equally regardless of location
- San Francisco salaries without San Francisco cost of living = major upside
- Equity can be significant at startups
- Benefits vary - check healthcare, 401k, PTO policies

### Best Practices for Remote Applications
1. **Optimize LinkedIn** - Set location to "Remote" or target city
2. **Build in public** - Blog, open-source, Twitter presence helps
3. **Network remotely** - Join remote work communities, Slack groups
4. **Show remote competence** - Highlight past remote/async work
5. **Apply directly** - Company sites > job boards for remote roles
